# Remotely — Docker Setup

> ⚠️ **FOR DEMO / TESTING PURPOSES ONLY — NOT INTENDED FOR PRODUCTION USE.**

Automated installer for [Remotely](https://github.com/immense/Remotely) — a self-hosted remote desktop and support tool accessible through a browser.

**Made by:** Mohammed Ali Elshikh — [prismatechwork.com](https://prismatechwork.com)

---

## 🛠 Usage

```bash
chmod +x remotely-ubuntu.sh
sudo bash remotely-ubuntu.sh
```

## 🌐 Access

| URL | `http://<server-ip>:5000` |
|-----|--------------------------|

On first visit, register your admin account.

## 🌐 Ports

| Port | Purpose |
|------|---------|
| `5000` | Remotely Web UI |
