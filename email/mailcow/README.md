## 📖 What is Mailcow?

Mailcow is a complete self-hosted email server suite. It bundles all the components needed to run a full mail server — SMTP, IMAP, spam filtering, webmail, and an admin panel — into a single Docker Compose stack.

## ✨ Features

- Postfix (SMTP) for sending and receiving email
- Dovecot (IMAP/POP3) for mail storage
- SOGo webmail and CalDAV/CardDAV
- Rspamd antispam with DKIM signing
- ClamAV antivirus
- Web admin panel for managing domains and mailboxes
- Let's Encrypt SSL auto-renewal
- Two-factor authentication

| `25` | TCP | SMTP (incoming mail) |
| `587` | TCP | SMTP submission (outgoing mail) |
| `465` | TCP | SMTPS |
| `143` | TCP | IMAP |
| `993` | TCP | IMAPS |

## ⚠️ DNS Requirements

For real email delivery you need proper DNS records:
- **MX record** pointing to your mail server
- **SPF record** for your domain
- **DKIM record** (generated by Mailcow)
- **DMARC record**
- **PTR / Reverse DNS** set by your hosting provider
