# Kavita — Docker Setup

> ⚠️ **FOR DEMO / TESTING PURPOSES ONLY — NOT INTENDED FOR PRODUCTION USE.**

Automated installer for [Kavita](https://www.kavitareader.com/) — a self-hosted digital library for manga, comics, and books (ePub, PDF, CBZ, CBR).

**Made by:** Mohammed Ali Elshikh — [prismatechwork.com](https://prismatechwork.com)

---

## 🛠 Usage

```bash
wget https://raw.githubusercontent.com/mhmdali94/Docker/main/media/kavita/kavita-ubuntu.sh
chmod +x kavita-ubuntu.sh
sudo bash kavita-ubuntu.sh
```

## 🔑 Credentials

| Field | Value |
|-------|-------|
| Admin account | Created on first visit |

## 🌐 Ports

| Port | Purpose |
|------|---------|
| `5000` | Kavita Web UI |

## 💻 Connect

```bash
# Web UI
http://SERVER_IP:5000

# Add your library files to
/root/docker/kavita/library/
```
