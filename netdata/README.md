# Netdata — Docker Setup

> ⚠️ **FOR DEMO / TESTING PURPOSES ONLY — NOT INTENDED FOR PRODUCTION USE.**

Automated installer for [Netdata](https://www.netdata.cloud/) — real-time performance and health monitoring for servers, containers, and applications.

**Made by:** Mohammed Ali Elshikh — [prismatechwork.com](https://prismatechwork.com)

---

## 🛠 Usage

```bash
chmod +x netdata-ubuntu.sh
sudo bash netdata-ubuntu.sh
```

## 🌐 Access

| URL | `http://<server-ip>:19999` |
|-----|---------------------------|

No login required — dashboard loads directly.

## 🌐 Ports

| Port | Purpose |
|------|---------|
| `19999` | Netdata Web Dashboard |
