# Navidrome — Docker Setup

> ⚠️ **FOR DEMO / TESTING PURPOSES ONLY — NOT INTENDED FOR PRODUCTION USE.**

Automated installer for [Navidrome](https://www.navidrome.org/) — a self-hosted music server and streamer, compatible with Subsonic/Airsonic clients.

**Made by:** [prismatechwork.com](https://prismatechwork.com)

---

## 🛠 Usage

```bash
chmod +x navidrome-ubuntu.sh
sudo bash navidrome-ubuntu.sh
```

## 🌐 Access

| URL | `http://<server-ip>:4533` |
|-----|--------------------------|

On first visit, create your admin account.

## 📂 Add Your Music

Place music files in:
```
/root/docker/navidrome/music/
```

## 🌐 Ports

| Port | Purpose |
|------|---------|
| `4533` | Navidrome Web UI |
