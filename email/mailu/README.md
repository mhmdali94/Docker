## 📖 What is Mailu?

Mailu is a simple yet full-featured self-hosted mail server. It is designed to be easy to set up and maintain, while providing all the components needed for a production-ready email server.

## ✨ Features

- Postfix (SMTP) for sending and receiving
- Dovecot (IMAP) for mail storage
- Roundcube webmail interface
- Rspamd antispam with DKIM support
- Admin web panel for domain and mailbox management
- Simple configuration via a single `.env` file
- Optional Let's Encrypt TLS

## 🌐 Access

| Admin UI | `http://<server-ip>/admin` |
|----------|---------------------------|
| Webmail | `http://<server-ip>/roundcube` |
| Username | `admin@example.com` |
| Password | Auto-generated during install (shown in terminal) |

> ⚠️ Change the admin password immediately after first login.

## 🔌 Ports

| Port | Protocol | Purpose |
|------|----------|---------|
| `80` / `443` | TCP | Web UI |
| `25` | TCP | SMTP (incoming mail) |
| `587` | TCP | SMTP submission |
| `465` | TCP | SMTPS |
| `143` | TCP | IMAP |
| `993` | TCP | IMAPS |

## ⚠️ DNS Requirements

For real email delivery you need proper DNS records:
- **MX record** pointing to your mail server
- **SPF record** for your domain
- **DKIM record** (generated by Mailu)
- **DMARC record**

## 📁 Directory Structure

```
/root/docker/mailu/
├── docker-compose.yml
├── mailu.env        # All configuration
├── mail/            # Mailboxes
├── filter/          # Rspamd data
├── dkim/            # DKIM keys
└── webmail/         # Roundcube data
```
