# Immich — Docker Setup

> ⚠️ **FOR DEMO / TESTING PURPOSES ONLY — NOT INTENDED FOR PRODUCTION USE.**

Automated installer for [Immich](https://immich.app/) — a self-hosted photo and video management solution with AI-powered face recognition and search.

**Made by:** Mohammed Ali Elshikh — [prismatechwork.com](https://prismatechwork.com)

---

## 🛠 Usage

```bash
wget https://raw.githubusercontent.com/mhmdali94/Docker/main/media/immich/immich-ubuntu.sh
chmod +x immich-ubuntu.sh
sudo bash immich-ubuntu.sh
```

## 🔑 Credentials

| Field | Value |
|-------|-------|
| Admin account | Created on first visit |

| Port | Purpose |
